Interface

Présentation des deux fenêtres d'interface

Interface de la fenêtre de démarrage permettant l'affection de variables

  • du nombre de pions (nbpions)
  • du nombre de pions (nbpions)
  • du nombre de couleurs (nbcolonnes)
  • du nombre de parties (nbparties)
  • de définir les noms des joueurs (listejoueur)

Plateau de jeu

Le plateau de jeu se décompose en un espace de jeu (partie haute) et un espace de commandes (parties basses).

Lancement du jeu

Au lancement du jeu, l'interface permet de définir le nombre de couleurs, le nombre de pions et le nombre de coups à jouer ainsi que les noms de deux joueurs.

Choix des couleurs

Le premier joueur (nommé codeur) choisit le code secret.

Choix des couleurs

Le deuxième joueur (nommé décodeur) tente de deviner le code secret. Le programme affiche un feedback composé de pions noirs et de pions blancs. Un pion noir signifie qu'un pion est bien placé. Un pion blanc signifie qu'un pion est de la bonne couleur, mais mal placé.

Manches

Les joueurs jouent deux manches chacun. A la fin de la partie, le programme indique le gagnant ou si le joueurs sont ex aequo.
